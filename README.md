# 🎬 YouTube Clone - Projeto DIO

Este projeto é um **clone da interface do YouTube**, desenvolvido como atividade prática durante o curso da **Digital Innovation One (DIO)**.

O objetivo foi aplicar na prática conceitos de **HTML5 e CSS3**, com foco em:

* Estruturação semântica
* Organização de layout com Flexbox
* Estilização profissional
* Tema escuro (Dark Mode)
* Boas práticas de código

---

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* Flexbox

---

## 🎯 Objetivo do Projeto

Reproduzir a interface de um player de vídeo semelhante ao YouTube, contendo:

* 🔝 Header com logo, barra de pesquisa e ícones
* 🎥 Seção principal com vídeo (imagem simulando vídeo)
* 📺 Informações do canal
* 🔴 Botão de inscrição
* 📂 Sidebar com vídeos recomendados
* 🌑 Tema totalmente escuro

---

## 🧠 Conceitos Aplicados

Durante o desenvolvimento foram praticados:

* Estrutura semântica (`header`, `main`, `div`)
* Reset CSS
* `box-sizing: border-box`
* Layout em duas colunas
* `display: flex`
* `justify-content` e `align-items`
* `flex: 1` e `flex: 2`
* `gap`
* `margin` e `padding`
* `border-radius`
* Pseudo-classes como `:hover`
* Organização e separação de responsabilidades (HTML estrutura / CSS estilo)
